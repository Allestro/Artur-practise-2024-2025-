Это телеграмм бот позволяющий пользователю задать любой вопрос касающийся "Аналитического центра Нижнего Новгорода" и при наличие информации в папке с документами upload дать пользвателю ответ на его вопрос с помощью ChatGPT
Для его использования необходимы: 
1.API любой нейоросети которая будет использоваться для формирования ответа пользвателю
2.Токен активного телеграмм бота
3.Файла с информацией для заполнения папки upload 
4.Указания модели поведения для нейросети
Примечание: в фале config.yaml я удалил токен телеграмм бота и API key нейросети, так как это личная информация.
